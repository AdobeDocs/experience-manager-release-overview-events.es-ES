# Actualización de la versión 2022.6.0 de AEM as a Cloud Service.

Obtenga información del equipo de productos de AEM y obtenga información sobre las funciones y las innovaciones de la última versión de Adobe Experience Manager [!DNL Assets], [!DNL Assets Essentials], [!DNL Sites], [!DNL Commerce Integration Framework], [!DNL Forms] y [!DNL Cloud Manager].

**Versión preliminar**

   * Entrega de imágenes nativas y optimizadas para la web en la nube
